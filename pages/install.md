## 1. Install Prerequisite

VEDA requires Node.js for some features (server mode, OSC, etc).
Please install Node.js before using VEDA.

https://nodejs.org/


## 2. Install VEDA

Then install VEDA from Atom's `Settings` view or by `apm` like this:

```bash
apm install veda
```
