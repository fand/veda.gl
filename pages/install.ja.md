## 1. 事前準備

VEDAの機能をフルに使うには、Node.jsをインストールする必要があります。
お好みの手段で事前にインストールしてください。

https://nodejs.org/ja/


## 2. VEDA本体のインストール

VEDAをAtomの `Settings` タブか、`apm` からインストールできます。

```bash
apm install veda
```
